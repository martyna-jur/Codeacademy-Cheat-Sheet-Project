Common CSS properties

Property Description Example
Font-family Specifies the font family for text p. {
font-family: "Times New Roman", Times, serif;
}
Font-size Specifies the font size of text h1 {
font-size: 45px;
}
Font-weight Specifies how bold or thin text appears p.thick {
font-weight: bold;
}
Text-align Specifies the horizontal alignment of text body {
text-align: center;
}
Color Sets the color of text .text {
color: pink;
}
Background-color Specifies the background color of an element * {
background-color: darkgrey;
}
Opacity Sets the opacity level for an element .footer {
opacity: 0.5;
}
Background-image Specifies one or more background images for an element .main-banner {
background-image: url('https://www.example.com/image.jpg');
}
!important Sets an importance of a property. Once !important is used, it is very hard to override. p {
color: blue !important;
}
